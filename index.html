⚙️ Configurações de IA
Como obter sua chave de API
1. Acesse console.anthropic.com
2. Crie uma conta e vá em API Keys → Criar chave
3. Copie a chave e cole abaixo — fica salva só neste navegador

✅ Créditos gratuitos ($5) são concedidos automaticamente para novos usuários.
Custo ~$0,003 por documento lido. A IA usa um proxy seguro para contornar restrições do navegador.
⚡ Para usar no navegador (fora do claude.ai):
1. Baixe o arquivo iniciar_proxy_IA.py (junto com o gerador)
2. Execute-o com dois cliques (ou python3 iniciar_proxy_IA.py)
3. Mantenha a janelinha aberta enquanto usa o gerador
4. Configure sua chave acima e clique em Testar conexão

Dentro do claude.ai a IA funciona automaticamente, sem proxy e sem chave.
